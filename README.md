# xavic-f.github.io

Personal website hosted on Vercel.

## Development

```bash
npm install
npm run dev
```

Site runs at http://localhost:3000

## Commands

- `npm run dev` - Local development server
- `npm run dev:vercel` - Vercel dev server (requires `vercel login`, supports cleanUrls)
- `npm run build` - Generate metadata

## Deployment

Automatically deployed to Vercel on push to main. Clean URLs enabled (e.g., `/work` instead of `/work.html`).
